Протокол - это набор соглашений, правил, по которым разные программы могут обмениваться информацией
Протокол http используется для взаимодейсвия браузера с веб-сервером
Протокол http сущесвует поверх стека протоколов tcp/ip

Дело в том, что соединение с сервером идёт по tcp/ip, а не по http. 
http начинает работать уже после того, как было установлено tcp соединение.
